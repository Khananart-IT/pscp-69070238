2.ความเข้าใจโจทย์ของฉัน

อธิบายโจทย์: ในโจทย์นี้จะให้เขียนโปรแกรมแสดงถึงคำนวณว่าบ้านในพิกัดนี้ จะถูกหมึกท่วมในกี่วินาทีจากการการขยายตัวของน้ำหมึกเป็นวงกว้าง

Input: 1.จำนวนพื้นที่ที่น้ำหมึกขยายไปต่อวินาทีและจำนวนคำร้องของผู้คน
       2.พิกัดของบ้านของคนที่ส่งคำร้อง (หากมีเพิ่มจะทำการส่ง input ลงในบรรทัดใหม่) 
Output: จำนวนวินาทีที่พิกัดบ้านของผู้ส่งคำร้องจะถูกน้ำหมึกท่วม (หากมีเพิ่มจะทำการส่ง output ลงในบรรทัดใหม่)

Constraints: ถ้าเวลาที่ท่วมพอดีไม่เป็นจำนวนเต็มให้คิดเป็นวินาทีถัดไป เช่น 2.14 เป็น 3 วินาที

3.แผนแรกของฉัน

ขั้นแรก: สร้างฟังก์ชันสำหรับการหาค่าต่างๆที่โจทย์ต้องการ โดยจะให้มีการ input อยู่ในฟังก์ชันข้างในและแยกแต่ละตัวใน input ลงในตัวแปรใหม่ด้วย split()

ขั้นที่สอง: สร้างตัวแปร radiant จากการคำนวณของตัวแปร area สำหรับการคำนวณระยะทางที่หมึกขยายตัวต่อ 1 วินาที

ขั้นที่สาม: สร้าง for loop สำหรับข้อมูลพิกัดพื้นที่ของบ้านผู้ส่งคำร้องและคำนวณว่าหมึกจะขยายท่วมบ้านของผู้ส่งคำร้องที่กี่วินาทีจาก
การใช้พิกัด x หรือ y หารกับ radiant เพื่อหาเวลา (วินาที)

ขั้นที่สี่: แสดง output เป็นเวลา (วินาที) ของแต่บ้านแต่ละคนที่ส่งคำร้องมา
