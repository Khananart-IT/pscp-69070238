ขั้นที่สอง: สร้างเงื่อนไขที่ว่าหาก input ของหน่วยอุณหภูมิในบรรทัดที่สองเป็นแบบนี้ ต้องคำนวณแบบใดโดยหากหน่วยอุณหภูมิที่ใส่ใน input ที่สองเข้ามาไม่ใช่ C (องศาเซลเซียส) จะทำการแปลงหน่วยนั้นให้เป็น C (องศาเซลเซียส) ก่อนแล้วจึงนำไปคำนวณหาหน่วยอื่น (ถ้ามี)

ขั้นที่สาม: สร้างสมการการคำนวณหน่วยอุณหภูมิ

ขั้นที่สี่: แสดง output ของค่าของหน่วยอุณหภูมิที่ต้องการโดยมีทศนิยม 2 หลัก
